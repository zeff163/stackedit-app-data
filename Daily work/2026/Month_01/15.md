**C++ IDE应用推荐**
IDE包含了代码【编辑、调试、编译】的功能，比起一般的C++编辑器（VS Code）无需额外配置编译环境，和使用gcc来编译。
https://zhuanlan.zhihu.com/p/20753388965

| IDE | 适用场景 | 优点 | 缺点 |
| --- | --- | --- | --- |
| CLion（推荐） | 大型 CMake 项目 | 强大代码分析、内置 GDB/LLDB 调试 | 付费（学生可免费）且需较大内存 |
| Visual Studio | Windows 开发 | MSVC、图形化调试、强大 UI | 仅适用于 Windows |
| Qt Creator | Qt GUI、嵌入式开发 | 适合 Qt 项目、轻量 | 非 Qt 项目支持较弱 |
| Eclipse CDT | 复杂 C++ 项目 | 免费、支持插件 | 运行较慢，界面老旧 |
| Code::Blocks | 轻量级 IDE | 运行快，易用 | 功能较弱，不适合大项目 |

> 其他参考：
> https://www.zhihu.com/answer/1976249567793148944
> https://www.zhihu.com/answer/120460183528


### Visual Studio使用
**visual studio 安装**
安装时在第四栏的【安装位置】处有3个需要选择的目录
+ 产品：软件本体存放位置
	> 官方说明：此位置用于存放特定于此版本产品的
文件。如果拥有固态系统驱动器，我们建议你接受默认位置，以便更快地运行产品。
已修改为：E:\Program_file\Visual studio\Visual studio

+ 下载缓存：更新下载时，新安装包的存放位置
	> 官方说明：此位置用于下载安装
文件。可选择在安装后不保留下载缓存,但如果更新、修改或重新安装产品的话,则需要再次下载这些文件。此位置只能在首次安装产品时才能设置。
已修改为：E:\Program_file\Visual studio\Package_a

+ 共享目录、工具和SDK：其他新增工作负荷的存放位置
	>  官方说明：此位置包含并排产品安装所需的共享文件。某些工具和SDK 安装到此驱动器上的某个位置，而其他一些可能替代此设置并安装到另一个驱动器。
已修改为：E:\Program_file\Visual studio\Share_a

还有一处系统缓存、工具和位置固定的sdk（大概1G大小）

更多安装细节参考
https://learn.microsoft.com/zh-cn/visualstudio/install/install-visual-studio?view=visualstudio
> 说明：一般Visual Studio安装需要20-50G，如果想缩减安装的大小，可以选择性的勾选安装内容，[参考链接](https://learn.microsoft.com/zh-cn/visualstudio/install/install-visual-studio?view=visualstudio#step-4---choose-workloads) 
>
> 目前已勾选：C++、Python、Java三种功能支持。可以通过Visual Studio Installer进行修改（建议右下角选择下载完成后安装）
> 
> 其中关于添加C++部分的功能及影响，参考 [AI回答](https://ima.qq.com/share/#/ai-chat/XyGEhk5S1).
> 需要指出的是，该部分内容即使修改了工作负荷的存放路径，也会安装在系统驱动器上（即软件所在硬盘的根目录）

+ **电脑剪切板**
	电脑剪切板通过Win+V打开（如果没有请在设置 > 系统 > 剪切板 >剪切板历史记录 中开启），最多保存25条记录，关机后自动清空。如果超出保存上限会清除旧记录，可以在剪切板中固定某条信息防止被清除。使用其他第三方剪切板工具，获得更大存储量和更多支持的文件类型，例如：ditto
	> 参考1：https://haokan.baidu.com/v?pd=wisenatural&vid=9490572042355260408
	>  视频保存：https://t.me/c/3528750793/6?thread=5
	> 参考2：https://www.160.com/article/8862.html


+ **WPS云空间**
	如何关闭云空间
	> WPS界面右上角寻找【设置】> 【文档云同步】，关闭即可

	如何批量删除云空间里面的文档
	> 直接在WPS应用的云空间处清理，会提示要求VIP才能批量删除。
	> 可以登录WPS云文档网页版 https://www.kdocs.cn/ 管理个人的云空间存档。在【最近】一栏可以手动逐个勾选项目（不推荐），也可以用shift快捷键批量勾选（和windows批量选取类似），然后直接删除即可。（【我的设备】处也有很多残余文件，记得清理）
	> 删除完成后进入左栏【回收站】彻底删除
	>
	> 参考：https://mr.baidu.com/r/1PVQ7RkPTTa?f=ot&u=10f97d0095c60433
	> 说明：云空间文件和我的云存档不是同一个东西，两者的存储空间是独立的。
