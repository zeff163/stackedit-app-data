#### 网络适配器故障解决过程
网络突然无法连接，右下角Wifi图标改变，无法检索到Wifi网络。一般这些情况出现在三种时候：1. 正在使用中突然消失 2. 电脑休眠后重新唤醒并消失 3. 关机后重新开机消失
检测发现问题通常是RealtekWLan网卡驱动丢失
> 通过win+X快捷键打开【设备管理器】，然后在网络适配器一栏看是否有这个网卡驱动或者网卡是否有黄色感叹号，如果异常则需要重新安装该驱动

个人猜测导致出现这个驱动失灵的原因有以下几个：
1. 驱动过时
由于现在Windows10已经不再更新，所以相关的驱动也都是旧版。自己在Thinkpad官网处下载了匹配机型的驱动，但是无法安装到电脑默认的自动检索的驱动文件夹，只能安装到其他的文件夹（即使用管理员权限也无法安装）
	```
	C:\Windows\System32\DriverStore\FileRepository
	```
2. 使用VPN
	一般出现问题之前的那次网络正常时，都用了VPN，可能是这个原因导致的。因此<font color =red>**建议休眠电脑前记得关掉VPN**</font>（完全把整个应用从后台清掉，如果不清掉会导致网络核心一直在运作，常常会出现该核心已经运转8h以上的情形），<font color =red>**关机前记得关掉VPN**</font>，防止VPN配置残留对驱动的影响。
	
3. 360网络修复导致
	360的搜索功能在VPN开启时是无法使用的，会提示网络异常（但可以输入链接打开网址）。如果上次关机时VPN未关闭，那么下次开机用360就会出现网络异常，然后需要用360浏览器自带的网络修复工具进行修复，频繁的修复也可能导致网卡驱动受到影响。

一般出现网络异常后该怎么操作？
1. 先在【设置】---【网络与Internet】---【状态】用【疑难解答】让电脑自行检测。
2. 如果疑难解答无效，那么可以【状态】下面使用【网络重置】，重启电脑并让电脑重新安装所有网卡驱动。
3. 如果重置无效，可以用【Win+X】打开【设备管理器】，在其中看是否是丢失了realtak驱动，如果丢失则重新安装。
	> 驱动位置在 `E:\Program_file` 默认安装在`C:\DRIVERS\WIN\WLAN`，<font color =red>**重新安装前记得删除掉这个文件夹防止旧驱动仍然残余的影响**</font> 

4. 如果以上方法都无效，建议多试几次，或者等一段时间再试，有时候可能第二天电脑网络自己就恢复正常了。如果长时间还未修复，建议找售后修理。

补充：
1. 在网络未恢复期间，如果需要电脑使用网络，<font color =red>**可以通过手机usb连接电脑实现共享wifi，但此时无法传输文档。**</font> 具体操作是：用数据线连接手机和电脑 ---> 打开手机设置【连接与共享】的【个人热点】（可以不用开启热点）---> 打开【USB网络共享】，之后电脑就会连上手机连上的wifi。
2. 上面的解决方式主要是针对realtek网卡驱动丢失或异常的情形，如果出现其他故障，可以去AI搜索
