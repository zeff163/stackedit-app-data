	  ### **分类说明**  
		- **编程语言**：直接用于编写程序或算法的语言（如Python、R）。  
		- **标记语言**：用于结构化文档或数据（如HTML、XML）。  
		- **数据交换格式**：用于存储或传输结构化数据（如JSON、YAML）。  
		- **图形/图表工具**：专注于图形生成或可视化（如SVG、Mermaid）。  
		- **脚本/自动化工具**：用于特定场景的脚本控制（如Bash、VBA）。

3. <a id ="03-2">[<font color = red> 跳回</font>](#03-1)
	GitHub Codespaces 的收费模式如下，具体以 **GitHub 官方最新政策**为准[官网](https://docs.github.com/zh/billing/managing-billing-for-your-products/managing-billing-for-github-codespaces/about-billing-for-github-codespaces)：
	### **1. 免费额度**
	- **个人免费账户（GitHub Free）**：  
	  每月可免费使用 **60 核心小时**（2 核实例 × 30 小时）和 **15 GB 存储空间**，适用于基础配置的 Codespace。
	- **GitHub Pro、Team 或 Enterprise 账户**：  
	  免费额度可能更高（如每月 180 核心小时），具体需查看订阅计划。

	### **2. 超出免费额度的收费**
	- **计算资源**：  
	  按**核心小时（Core-Hours）**计费，费用取决于实例配置（如 2 核、4 核等）。  
	  例如：  
	  - 2 核实例：$0.18/核心小时  
	  - 4 核实例：$0.36/核心小时  
	  （实际价格可能随地区或活动调整）
	- **存储空间**：  
	  超出免费额度后，按 **$0.07/GB/月** 计费（存储从 Codespace 创建开始持续计费）。

	### **3. 自动暂停机制**
	- Codespace 在 **30 分钟无活动后自动暂停**，停止计算资源计费（存储仍计费）。

	### **4. 企业级服务**
	- **GitHub Enterprise Cloud** 用户可定制团队或组织的 Codespaces 配额，费用通过企业协议结算。

	### **如何监控用量？**
	- 在 GitHub 账户的 **Billing 页面** 查看实时使用情况，设置用量提醒。

	### **建议**
	- 个人轻度使用通常可控制在免费额度内。  
	- 高频或高配置用户需关注用量，避免意外费用。  
	- 详细定价参考：[GitHub Codespaces 定价](https://docs.github.com/en/billing/managing-billing-for-github-codespaces/about-billing-for-github-codespaces)（需登录 GitHub 账户）。
