### <font color = red>寻找一个具有以下特点的存储工具：
* **具有较好的文本编辑功能**  
方便轻巧，功能全面，易于编写，最好能支持多种语言，例如：`Latex`、`C++`、`Markdown`等。
* **可以多设备使用**
例如：有手机端软件，也有pc端应用。
* **可以与其他软件对接**
文件导出后可以直接导入其他软件中，不会产生无法识别或复原的麻烦。
* **支持多种文件格式**
例如：图片、视频、txt、doc、pdf等文件，可以用来当图床。
* **线上与本地可以相互传输数据**
可以实时将本地段的文件或文本传到网上，实现不同平台的数据接收。
* **可以用来当资源库**
有比较大的容量，空间比较安全、稳定。
* **易于备份**
最好能实时备份。
* **可协同工作**
支持多人同时合作
