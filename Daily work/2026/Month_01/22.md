| 平台 | 一句话定位 |
| :--- | :--- |
| **Coze** | 面向大众的 AI Bot 快速生成器 —— 无需编码，10 分钟发布一个带知识库、插件、工作流的聊天机器人（如客服、营销助手）。 |
| **Dify** | 面向开发者的 AI 应用开发平台 —— 提供可视化 Prompt 编排、RAG、Agent、可观测性，用于构建企业级私有化 AI 应用。 |
| **n8n** | 面向工程师的通用自动化工作流引擎 —— 通过节点连接 500+ 应用（Slack、Google Sheets、数据库等），实现



| 能力 | Coze | Dify | n8n |
| :--- | :--- | :--- | :--- |
| **主要用途** | 快速创建聊天机器人（Bot） | 构建企业级 AI 应用（Web/API） | 自动化业务流程（Zapier 替代品） |
| **是否支持 LLM** | ✅ 内置大模型（可选 Claude/GPT/Qwen） | ✅ 支持私有模型（Ollama/OpenAI/国产 API） | ❌ 本身不处理 LLM，但可通过 HTTP 调用外部 LLM |
| **RAG（知识库）** | ✅ 拖拽上传文档，自动问答 | ✅ 强大 RAG 管理（分段、嵌入、召回优化） | ⚠️ 需手动集成（如调用 Dify API） |
| **Agent / 工作流** | ✅ 可视化插件编排（如查天气+发邮件） | ✅ 支持 ReAct、Plan-and-Execute 等 Agent 模式 | ✅ 核心优势：复杂条件分支、循环、错误重试 |
| **部署方式** | ❌ 仅 SaaS（字节云） | ✅ 支持私有化部署（Docker/K8s） | ✅ 开源 + 私有化部署（Node.js） |
| **API 能力** | ⚠️ 有限（主要用于 Bot 嵌入） | ✅ 提供完整 RESTful API，可集成到业务系统 | ✅ 所有工作流均可通过 Webhook 触发或提供 API |
| **编程扩展** | ❌ 无代码 | ✅ 支持 Python/JS 自定义函数（Tool） | ✅ 支持 JavaScript 表达式 + 自定义节点 |
| **典型用户** | 运营、产品经理、中小企业 | 企业架构师、AI 工程师、SaaS 厂商 | DevOps、IT 自动化工程师、业务分析师 |
